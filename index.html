We have been working on the development of this project for a year, financing it with our own funds and with entrepreneurship helping programs of the Basque Government.

Only the 2.2% of Web pages comply with the old WCAG 2.0 Web accessibility standard.
(source: WebAIM)

We want to remedy it by eradicating the most deeply rooted myths about Web accessibility:

"Web accessibility is not designing compatible, and accessible Web sites are not modern, they are simple and ugly."

"Web accessibility has no market."

"Creating accessible Web sites is very expensive and there are not enough qualified Web accessibility professionals."

What we have already done

In an effort to spread the importancy of Web accessibility we have developed three a11y utilities. We want to change the perspective that most people have about this matter so it is no longer treated as something expensive, unuseful or difficult. The time is now. All our creations are open-source and free to use, you can find them on GitHub.

PUXL
Framework

A front-end library for the modern Web. Create accessible, responsive Web User Interfaces with lightweight HTML following the WCAG 2.1 standarts.

Guarantee access to Web content for all the people around the world without deep knowledge in Web accessibility.

  • Collects and automates the existing theoretical resources.

  • Intuitive and a practically flat learning curve.

  • Visual, modular and scalable.

  • It is preventive it detects, alerts and suggests solutions to coding errors while developing.

  • Guaranteed WCAG 2.1 AA accessibility specification

  • To optimize the resulting code to the point that it reduces energy and data cost.

  • It guarantees and equal access regardless of our functional capabilities.

  • Holistic UX, focused on the developer, the customer, the end user and their experience.

Omnibus
Prime

Omnibus in latin means for all. Prime because is our first theme for WordPress.

A classic blog starter theme made using PUXL framework tailor-fitted for WordPress. Have your stories accessible, WCAG 2.1 AA compliant with zero effort: color-contrasted, responsive, customizable, human and eco-friendly.

PUXL
Icons

An icon library of more than 300 handmade and custom tailored which features an accessible title and description for each one of the icons.

Thanks to one of our collaborators you can use this package in React. Find it ready to use in npm and the source code un GitHub.

What we are doing now

PUXL Accessibility Zone is the starting point for making an universal and accessible Web. A space that puts tools available to centralize on a single platform all the effort that is made in Web accessibility worldwide.

We want to turn the accessibility into the new standard of web development on a global scale.

  • COMMUNITY

    A professional network where you can interact with the most prominent specialists in accessibility and Web development.

  • LEARN

    A series of courses in webinar and PDF format to acquire at your pace all the skills necessary for accesssible Web development using PUXL framework technology.

  • BLOG

    The space where you can consult the articles created by the community. You can also access the Accessibility First Show produced by the PUXL clan where we talk about Web accessibility and interview the best specialists in the world.

  • STORE

    Market of accessible themes and web templates, created with PUXL technology for the most relevant development frameworks and CMS on the market (WordPress, Drupal, Joomla …), as well as dedicated websites. It will also be the place where you can buy merchandise such as T-shirts, mugs…

Why 30.000€?

The real capital need that we have to achieve our objectives is € 86,263, which includes the personnel hiring expenses that will help us accelerate the PUXL Accessibility Zone development process. The decision to lower our collection target to 30,000 euros is because we know that even with the small team we are we will be able to carry out our product, albeit at a much lower speed. We have decided to make the possibility of reaching 100% of our budget an extended objective, giving this objective as a reward, the extension to double the premium subscription periods offered as a reward. We have complete confidence in our product and we believe that this is a good way to convey that confidence to you.

Ecological software. PUXL can have a big social impact beyond Web accessibility; a clear environmental impact. PUXL framework was created with the target to help developers create accessible, modern, robust and optimized web interfaces.

As a result of our techniques, both server-sided and client-side devices experience less consumption of resources on the loading speed of the Web, which meand less energetic impact made by these machines.

In addition to this, one of the PUXL framework premises is to guarantee that the Web developed is accessible in low favorable conditions, that is to say, with low-speed Internet connections of and/or with low hardware resources devices. A Web page made with PUXL framework would have more probabilities of being loaded correctly that any other Web page developed with other technologies. We realized that if we can keep using our devices for long we will not be needing a more powerful one, reducing the electronic waste.

In the day this article was written, according to information from Internet Live Stats, 54,6% of the world population, which means more than 4,5 billion people are Internet users, and they are surfing along with more than 1,7 billion active Web pages.

According to the Clicking Clean report of Greenpeace of 2017, about 7% of the planet energy consumption derives from the energy demand associated with what some call "digital capitalism"; the above-mentioned number has increased since the publication of this study, and it is believed that it will achieve 14% in 2040.

Can you imagine that if we will cause an average 10% positive impact as minimally on the environmental trace produced by the digital age?
